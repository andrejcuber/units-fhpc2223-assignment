# FHPC: Final assignment
This is repository for final assigment of Foundations for high perfomance computing class that was thought at University of Trieste in academic year 2022/2023 as part of the Data Science and Scientific Computing master degree.

Final assignment contained two exercises, which are stored each in their own directory with all its data in this repository. Additionaly to the two exercise folders, in the main directory we can find a report, that covers both exercises results.
